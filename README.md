# Web Scraping - Mineração de Dados

Projeto de coleta e análise de dados de mangás utilizando web scraping.

## Objetivo
Coletar aproximadamente 1000 registros de mangás e realizar tratamento e análise exploratória dos dados.

## Tecnologias
- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib

## Como executar
Instale as dependências:

```bash
pip install -r requirements.txt

python project.py
